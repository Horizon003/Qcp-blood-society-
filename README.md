🩸 QCP Blood Society App

QCP Blood Society App is a modern blood donation platform designed to connect donors and recipients efficiently. It enables users to request blood, find available donors, and contribute to saving lives through a fast and reliable system.

⸻

🚀 Features

* 🧑‍🤝‍🧑 Donor Registration System
* 🩸 Blood Request Submission
* 🔍 Search & Filter Donors by Blood Group
* ⚡ Real-time Database Integration
* 🔐 Secure Data Handling
* 📱 Responsive & User-Friendly Interface

⸻

🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend/Database: Firebase (Realtime Database / Firestore)
* Storage: Firebase Storage / Cloudinary
* Hosting: GitHub Pages / Hostinger
